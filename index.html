ADD Forest Species Segmentation in Aerial Imagery of Wasatch Mountains Using 3D Convolutional Neural Network
Joshua James Stapley Montoya
2023
implementation details / segmented map

Aerial imagery of the Wasatch Mountains was downloaded from the United States Geological Survey (USGS) EarthExplorer portal, taken as part of the National Agriculture Imagery Program (NAIP). Partial ground truth images were created using 6 of the images: 3 for training, 3 for testing. A 3D CNN model was trained using Hyperspectral Imaging (HSI) Cubes. Pixel classification accuracies on the 3 unseen test images were 84, 74, and 73 percent. Using this model, all 144 images in the dataset were segmented. A zoomable map containing all of this imagery can be found at this link. Project is ongoing.

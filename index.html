Forest Species Segmentation in Aerial Imagery of Wasatch Mountains Using 3D Convolutional Neural Network
Joshua James Stapley Montoya
2023
implementation details / zoomable segmented map

Aerial imagery of the Wasatch Mountain Range, provided by the National Agriculture Imagery Program (NAIP), was self-labeled for partial ground truth to train a model to segment all 144 images in the dataset by species ground cover. A 3D CNN model was trained using Hyperspectral Imaging (HSI) Cubes. Pixel classification accuracies on 3 unseen test images were 84, 74, and 73 percent. A zoomable map containing all of this imagery can be found at this link. Project is ongoing.
